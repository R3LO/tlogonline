# Исправление переполнения текста "Регистрация" в шапке

## Проблема
На мобильных устройствах и маленьких экранах текст "Регистрация" в навигационной панели не помещался по ширине, что приводило к переполнению и некорректному отображению.

## Решение
Добавлены специальные CSS стили для предотвращения переполнения текста в навигации:

### 1. Базовые стили переполнения
- Добавлен `overflow-x: hidden` для контейнера navbar
- Добавлены стили `overflow: hidden`, `text-overflow: ellipsis`, `white-space: nowrap` для навигационных ссылок
- Установлена максимальная ширина для ссылок навигации

### 2. Адаптивные стили для мобильных устройств
- **576px и меньше**: уменьшен размер шрифта до 0.8rem, максимальная ширина 100px
- **400px и меньше**: еще больше уменьшен размер шрифта до 0.7rem, максимальная ширина 70px

### 3. Специальные стили для ссылок "Регистрация" и "Вход"
- На экранах ≤576px: размер шрифта 0.75rem, максимальная ширина 80px
- На экранах ≤400px: размер шрифта 0.65rem, максимальная ширина 60px

## Измененные файлы
- `tlog/static/css/base.css` - основные стили навигации
- `tlog/static/css/simple-navbar.css` - упрощенные стили навигации

## Результат
Теперь текст "Регистрация" корректно отображается на всех размерах экранов без переполнения, с обрезкой длинного текста и многоточием.
