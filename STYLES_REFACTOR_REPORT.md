## Как использовать новые стили

### Основные классы
```html
<!-- Кнопки -->
<button class="btn btn-primary">Кнопка</button>
<button class="btn btn-outline-primary">Контурная</button>

<!-- Карточки -->
<div class="card">
    <div class="card-header">Заголовок</div>
    <div class="card-body">Контент</div>
</div>

<!-- Статистика -->
<div class="stat-card">
    <i class="stat-icon fas fa-radio"></i>
    <div class="stat-value">1,234</div>
    <div class="stat-label">Связей</div>
</div>

<!-- Градиентный текст -->
<h1 class="text-gradient">TLog</h1>

<!-- Анимации -->
<div class="fade-in">Появится с анимацией</div>
<div class="slide-up">Скользит вверх</div>
```

### CSS Переменные
Все цвета и размеры теперь управляются через переменные в `variables.css`:
```css
:root {
    --primary: #667eea;
    --shadow-md: 0 2px 8px rgba(0, 0, 0, 0.1);
    --radius-lg: 12px;
}
```
