# Структура шаблонов и внешних файлов для LoTW

## Созданные файлы

### Шаблоны
- `tlog/templates/lotw_base.html` - Базовый шаблон для страниц LoTW
- `tlog/templates/lotw.html` - Основная страница LoTW, наследующая от базового шаблона

### Стили (CSS)
- `tlog/static/css/base.css` - Базовые стили для всего проекта
- `tlog/static/css/lotw.css` - Специфичные стили для страницы LoTW

### Скрипты (JavaScript)
- `tlog/static/js/common.js` - Общие JavaScript функции для проекта
- `tlog/static/js/lotw.js` - JavaScript функциональность для страницы LoTW

## Иерархия наследования

```
base.html (главный базовый шаблон)
    ↓ наследует
lotw_base.html (базовый шаблон для LoTW страниц)
    ↓ наследует
lotw.html (конкретная страница LoTW)
```

## Структура загрузки ресурсов

### В base.html загружаются:
- Bootstrap CSS
- Font Awesome иконки
- common.css (общие стили проекта)
- base.css (базовые стили)
- common.js (общие скрипты)

### В lotw_base.html дополнительно загружаются:
- lotw.css (специфичные стили LoTW)

### В lotw.html дополнительно загружаются:
- lotw.js (специфичные скрипты LoTW)

## Особенности реализации

### Блочная структура
- `{% block lotw_content %}` - основной контент страницы LoTW
- `{% block extra_css %}` - дополнительные стили
- `{% block extra_js %}` - дополнительные скрипты

### Функциональность JavaScript

#### common.js предоставляет:
- Систему уведомлений
- AJAX утилиты с CSRF защитой
- Валидацию форм
- Инициализацию Bootstrap компонентов
- Утилиты для работы с датами

#### lotw.js предоставляет:
- Анимации карточек
- Обновление статуса LoTW
- Интерактивность быстрых действий
- Tooltips и модальные окна

### CSS архитектура

#### base.css включает:
- CSS переменные для цветовой схемы
- Улучшенные компоненты (кнопки, карточки, формы)
- Анимации и переходы
- Responsive дизайн
- Поддержку темной темы

#### lotw.css включает:
- Специфичные стили для карточек LoTW
- Hover эффекты
- Анимации появления
- Responsive улучшения

## Использование

### Создание новой LoTW страницы
1. Создайте новый файл, наследующий от `lotw_base.html`
2. Определите блок `{% block lotw_content %}`
3. При необходимости добавьте дополнительные стили в `{% block extra_css %}`
4. При необходимости добавьте дополнительные скрипты в `{% block extra_js %}`

### Пример:
```html
{% extends 'lotw_base.html' %}
{% load i18n %}
{% load static %}

{% block lotw_content %}
<div class="row">
    <div class="col-12">
        <div class="card fade-in">
            <div class="card-body">
                <h5>Новая LoTW страница</h5>
                <p>Контент здесь...</p>
            </div>
        </div>
    </div>
</div>
{% endblock %}

{% block extra_js %}
<script>
// Дополнительные скрипты для этой страницы
</script>
{% endblock %}
```

### Добавление новых общих компонентов
1. Для стилей - добавляйте в `base.css`
2. Для JavaScript - добавляйте в `common.js`
3. Для специфичных LoTW компонентов - добавляйте в соответствующие файлы

## Преимущества архитектуры

1. **Модульность** - четкое разделение ответственности
2. **Повторное использование** - общие компоненты доступны на всех страницах
3. **Наследование** - удобная иерархия шаблонов
4. **Производительность** - внешние файлы кэшируются браузером
5. **Поддержка** - легко добавлять новые функции и стили
6. **Responsive** - адаптивный дизайн для всех устройств
7. **Анимации** - плавные переходы и эффекты
8. **Доступность** - поддержка Bootstrap компонентов
