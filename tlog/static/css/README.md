# TLog - Система стилей

## Обзор
Полностью переработанная система стилей для проекта TLog, организованная по модульному принципу для лучшей поддержки и масштабируемости.

## Структура файлов

### Основные файлы
- **`main.css`** - Главный файл, который импортирует все остальные стили
- **`variables.css`** - CSS переменные (цвета, размеры, отступы и т.д.)
- **`base.css`** - Базовые стили для всех элементов
- **`components.css`** - UI компоненты (кнопки, карточки, формы и т.д.)
- **`layout.css`** - Макеты и позиционирование
- **`pages.css`** - Стили для конкретных страниц
- **`utilities.css`** - Вспомогательные классы и утилиты
- **`responsive.css`** - Адаптивные стили

## Использование

### Подключение стилей
В `base.html` подключается только один файл:
```html
<link href="{% static 'css/main.css' %}" rel="stylesheet">
```

### CSS Переменные
Все цвета, размеры и другие свойства определены в `variables.css` как CSS переменные:
```css
:root {
    --primary: #667eea;
    --primary-hover: #5a67d8;
    --secondary: #764ba2;
    --shadow-md: 0 2px 8px rgba(0, 0, 0, 0.1);
    --radius-lg: 12px;
    --padding-md: 1rem;
}
```

### Основные компоненты

#### Кнопки
- `.btn` - базовая кнопка
- `.btn-primary` - основная кнопка
- `.btn-outline-primary` - контурная кнопка
- `.btn-sm` - маленькая кнопка

#### Карточки
- `.card` - базовая карточка
- `.stat-card` - карточка статистики
- `.utility-card` - утилитарная карточка

#### Навигация
- `.navbar` - навигационная панель
- `.nav-link` - ссылка навигации

#### Формы
- `.form-control` - поле ввода
- `.form-select` - выпадающий список
- `.form-label` - подпись к полю

#### Утилиты
- `.text-gradient` - градиентный текст
- `.bg-gradient` - градиентный фон
- `.shadow-md` - средняя тень
- `.rounded-lg` - сильное скругление
- `.fade-in` - анимация появления
- `.slide-up` - анимация скольжения вверх

## Адаптивность

### Брейкпоинты
- **xs**: < 576px (мобильные)
- **sm**: ≥ 576px (малые экраны)
- **md**: ≥ 768px (планшеты)
- **lg**: ≥ 992px (десктопы)
- **xl**: ≥ 1200px (большие экраны)

### Классы адаптивности
```css
.d-xs-none { display: none !important; }
.d-sm-block { display: block !important; }
.d-md-flex { display: flex !important; }
```

## Тема

### Цветовая схема
- **Основные цвета**: `--primary`, `--secondary`
- **Статусные цвета**: `--success`, `--warning`, `--danger`, `--info`
- **Нейтральные цвета**: `--light`, `--dark`, `--muted`
- **Градиенты**: `--bg-gradient`

### Градиенты
```css
.bg-gradient {
    background: linear-gradient(135deg, var(--primary), var(--secondary));
}
```

## Анимации

### Доступные анимации
- `.fade-in` - появление
- `.slide-up` - скольжение вверх
- `.slide-down` - скольжение вниз
- `.slide-left` - скольжение влево
- `.slide-right` - скольжение вправо
- `.bounce-in` - упругое появление
- `.pulse` - пульсация
- `.shake` - тряска

### Задержки анимаций
- `.delay-1` - задержка 0.1s
- `.delay-2` - задержка 0.2s
- `.delay-3` - задержка 0.3s
- `.delay-4` - задержка 0.4s
- `.delay-5` - задержка 0.5s

## Примеры использования

### Основная кнопка
```html
<button class="btn btn-primary">
    <i class="fas fa-save"></i>
    Сохранить
</button>
```

### Карточка статистики
```html
<div class="stat-card">
    <i class="stat-icon fas fa-radio"></i>
    <div class="stat-value">1,234</div>
    <div class="stat-label">Связей</div>
</div>
```

### Адаптивная сетка
```html
<div class="row gap-3">
    <div class="col-md-6">
        <div class="card">
            <div class="card-header">Заголовок</div>
            <div class="card-body">Контент</div>
        </div>
    </div>
</div>
```

### Градиентный текст
```html
<h1 class="text-gradient">TLog</h1>
```
