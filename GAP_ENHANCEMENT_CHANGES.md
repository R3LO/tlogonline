# Увеличение базового gap для больших экранов

## Описание изменений

Внесены изменения в CSS для увеличения базового gap (расстояния) для основного пункта меню на больших экранах.

## Изменённые файлы

### 1. `tlog/static/css/variables.css`
- **Добавлены новые CSS переменные:**
  - `--padding-2xl: 3rem` (48px)
  - `--padding-3xl: 4rem` (64px)

### 2. `tlog/static/css/layout.css`
- **Расширены gap утилиты:**
  - Добавлены `.gap-5` и `.gap-6` классы для использования новых переменных

### 3. `tlog/static/css/responsive.css`
- **Добавлены медиа-запросы для больших экранов:**

#### Для экранов 1400px+ (Extra Extra Large):
- Увеличен padding для `.nav-link`: `var(--padding-md) var(--padding-lg)`
- Увеличены gap утилиты:
  - `.gap-1`: `var(--padding-md)` (16px)
  - `.gap-2`: `var(--padding-lg)` (24px) 
  - `.gap-3`: `var(--padding-xl)` (32px)
  - `.gap-4`: `3rem` (48px)
- Увеличен gap для `.navbar-nav`: `var(--padding-md)`
- Увеличен padding для `.btn`

#### Для экранов 1600px+ (Ultra Large):
- Еще больше увеличен padding для `.nav-link`: `var(--padding-lg) var(--padding-xl)`
- Максимальные gap утилиты:
  - `.gap-1`: `var(--padding-lg)` (24px)
  - `.gap-2`: `var(--padding-xl)` (32px)
  - `.gap-3`: `3rem` (48px)
  - `.gap-4`: `4rem` (64px)
- Увеличен gap для `.navbar-nav`: `var(--padding-lg)`
- Увеличен padding и font-size для `.btn`

#### Новые адаптивные gap классы:
- `gap-xl-1` до `gap-xl-4` для экранов 1400px+
- `gap-xxl-1` до `gap-xxl-4` для экранов 1600px+

## Результат

- **На больших экранах (1400px+)**: основные пункты меню становятся более заметными с увеличенными отступами
- **На ультра больших экранах (1600px+)**: еще большее увеличение для оптимального восприятия
- **Адаптивность**: изменения применяются только на соответствующих размерах экранов

## Использование

Для принудительного использования увеличенных gap на больших экранах можно использовать новые классы:
- `gap-xl-2` для экранов 1400px+
- `gap-xxl-3` для экранов 1600px+

Примеры:
```html
<!-- Автоматически увеличивается на больших экранах -->
<div class="d-flex gap-2">
    <button>Кнопка 1</button>
    <button>Кнопка 2</button>
</div>

<!-- Принудительно увеличенный gap для 1400px+ -->
<div class="d-flex gap-xl-3">
    <button>Кнопка 1</button>
    <button>Кнопка 2</button>
</div>
```
